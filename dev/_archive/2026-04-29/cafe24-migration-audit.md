## 4. 디자인/UX 비교

### 4-1. stiz.kr (카페24) UI 특성

- **다크 테마** 기반 (검정 배경, 흰색 텍스트가 주조)
- 대형 **이미지 중심** 레이아웃 — 히어로 배너, 상품 이미지 강조
- 카페24 스마트디자인 기반으로 **표준적인 쇼핑몰 UX** (메뉴 > 리스트 > 상세 > 장바구니 > 결제)
- 모바일 반응형은 카페24 기본 제공 수준
- **폰트**: 한글 고딕 계열, 영문 산세리프
- **컬러**: 검정/흰색 위주, 액센트 컬러 최소화
- 메뉴 구조가 **수평 드롭다운** (STIZ / TEAMWEAR / STORE / COMMUNITY)

### 4-2. 자체 사이트 현재 UI

- **화이트 테마** 기반 (흰색 배경, 검정 텍스트)
- 메인 페이지만 히어로 영역에서 다크 (스크롤 시 화이트 헤더로 전환)
- **Tailwind CSS** 기반 유틸리티 스타일링 — 일관된 디자인 시스템
- **Material Symbols** 아이콘 통일
- **Pretendard** 한글 폰트 (현대적)
- 메가 메뉴 (TEAMWEAR 3단 구성: 카테고리 + 디자인서비스 + 프로모 이미지)
- **브랜드 컬러**: black(#111111) + red(#E63946)
- 반응형: Tailwind 브레이크포인트 (sm/md/lg)로 모바일 대응
- **자체 차별화**: AI 디자인 랩, FOMO 토스트, 인스타 피드, AI 챗봇

### 4-3. 차이점 요약

| 항목 | stiz.kr | 자체 사이트 | 평가 |
|------|---------|-----------|------|
| 테마 | 다크 | 화이트 | 자체가 더 현대적이고 깔끔 |
| 메뉴 | 기본 드롭다운 | 메가메뉴 + 동적 카테고리 | 자체가 우수 |
| 모바일 | 카페24 기본 | Tailwind 반응형 | 동등 |
| 상품 상세 | 표준 쇼핑몰 | 사이즈 추천 + 커스텀 인라인 | 자체가 우수 |
| 결제 | PG 연동 완비 | 계좌이체만 | stiz.kr이 우수 |
| 검색 | 카페24 검색엔진 | API 기반 오버레이 | 동등 |
| 게시판 | 완비 (공지/문의/후기) | 공지+문의만 (정적) | stiz.kr이 우수 |
| 관리자 | 카페24 어드민 | 자체 14페이지 | 자체가 훨씬 우수 |

### 4-4. 개선 방향

- 테마는 현재 화이트 유지 (현대적 트렌드에 부합)
- stiz.kr의 **이미지 중심 레이아웃** 장점은 이미 메인 페이지에 반영됨
- **결제 시스템**이 가장 큰 격차 — PG 연동이 핵심 과제
- **게시판 시스템** 구축으로 공지/문의/후기 3종 DB 연동 필요

### 핵심 격차 3가지

1. **결제**: PG 미연동 (카드/간편결제 불가)
2. **게시판**: 리뷰/커뮤니티 시스템 없음
3. **인증**: 서버 기반 인증 미전환 (보안 이슈)

### 자체 사이트 우위 3가지

1. **AI 디자인 랩**: 2D/3D 커스텀 도구 (stiz.kr에 없는 차별화)
2. **관리자 시스템**: 14페이지 자체 어드민 (카페24 어드민보다 맞춤형)
3. **주문 관리**: 13단계 상태 추적 + 4파트 체계 (카페24 기본보다 강력)
