5. **멱등성 0** — 같은 status 두 번 클릭 시 두 번 발송됨.
6. **`server/.env.example`과 루트 `.env.example` 중복** — server는 `server/.env`만 로드 → **루트 .env가 아닌 server/.env에 키** 함정.

## D. 사용자 사전 준비 체크리스트 (비개발자용)

> **비유**: SOLAPI는 우체국 대행. STIZ가 편지 쓰고 SOLAPI가 카카오톡으로 배달. 카카오는 광고/스팸 막으려고 "이 편지 양식 검토받았어?"를 요구 → **템플릿 사전 등록 + 심사**

### 단계 1. SOLAPI 가입 (5분)
1. https://solapi.com → "회원가입"
2. 사업자 정보 입력 (STIZ 사업자등록증 필요)
3. 본인인증(휴대폰) 완료

### 단계 2. 발신번호 등록 (10분)
1. 콘솔 → "발신번호 관리" → "발신번호 등록"
2. 070-4337-3000 (또는 등록 가능 번호)
3. 사업자등록증 업로드 → 1~2시간 내 승인

### 단계 3. API 키 발급 (3분)
1. 콘솔 → 우상단 프로필 → "API Key 관리"
2. "API Key 생성" → API Key, API Secret 복사 (Secret 다시 못 봄)
3. 권한: "메시지 발송" 체크

### 단계 4. 카카오 비즈니스 채널 연결 (1~3일 — 가장 큼)
1. 콘솔 → "카카오 알림톡" → "채널 추가"
2. 카카오 비즈니스 채널 ID(@stiz) 입력 — 없으면 https://business.kakao.com 에서 먼저 생성
3. 카테고리 심사 — 최대 3영업일

### 단계 5. 알림톡 템플릿 등록 + 카카오 심사 (1~3일 — 7종 × 3일 가능)
1. 콘솔 → "카카오 알림톡" → "템플릿 등록"
2. 7종 템플릿 본문을 **이 문서 Part 2 E 섹션**에서 복사 (변수는 `#{변수명}`)
3. 각 템플릿 카카오 심사 — 2~3영업일
4. 거절 시 사유에 따라 본문 수정 후 재심사
5. 승인되면 콘솔에 PFID(채널) + templateId 부여 → 코드에 입력

### 단계 6. 환경변수 입력 (2분)
- `server/.env` 파일에 (⚠️ **루트 `.env`가 아니라 `server/.env`** — server.js가 거기서 로드)
```
SOLAPI_API_KEY=NCSXXXXXXXXXXXX
SOLAPI_API_SECRET=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
SOLAPI_SENDER_PHONE=07043373000
SOLAPI_KAKAO_PFID=KA01PF...    ← Phase 1에서 추가
```

### 단계 7. 충전금 (5분)
- 콘솔 → "충전" → 신용카드 충전
- **첫 한달 예상**: 일평균 5건 × 7종 = 일 35건, 월 1,050건. 알림톡 9원 × 1,050 = **약 1만원**. 5만원 충전 권장(여유분).

## C. 카카오 심사 거절 위험 진단

> **비유**: 카카오 알림톡은 "공공기관 안내문" 톤만 허용. 광고 메일 톤(`[브랜드]`, `★`, `이벤트`, `할인`)은 거절. 우리 7종 모두 첫 줄에 `[STIZ]` → 광고성 대괄호로 분류 위험.

| # | 알림 종류 | 위험도 | 위험 요인 |
|---|----------|--------|----------|
| 1 | order_created | 🟡 중간 | `[STIZ]` 대괄호 / "정상적으로" 가벼운 톤 |
| 2 | design_confirmed | 🟡 중간 | `[STIZ]` 대괄호 |

## G. 운영 가이드

### G-1. SOLAPI 콘솔 템플릿 등록 위치
1. https://console.solapi.com 로그인
2. 좌측 메뉴 → "카카오 알림톡" → "알림톡 템플릿"
3. 우측 상단 "+ 템플릿 등록"
4. 채널(PFID) 선택 → 종류(정보성) → 본문 입력 → 변수 명시
5. "승인 요청" → 카카오 심사 (2~3영업일)

### G-2. 변수 형식 (카카오 표준)
- ✅ `#{고객명}` (중괄호 + 우물 정자)
- ❌ `${고객명}` (JS 형식)
- ❌ `{{고객명}}` (다른 시스템)
- ❌ `#{customer.name}` (점 표기 불가)

### G-3. 거절 시 대응
1. SOLAPI 콘솔 → 해당 템플릿 → "거절 사유" 확인
2. 흔한 사유 3가지:
   - "광고성 표현 포함" → `[STIZ]` 대괄호 제거
   - "정보성 알림 부적합" → 거래/안내 명확하게
   - "변수 예시 미흡" → 변수마다 예시 입력
3. "재심사 요청" → 2~3영업일
4. 같은 템플릿 3회 거절 시 30일 재신청 제한

### G-4. 한 번 통과한 본문은 수정 불가
- 승인 후 **본문 한 글자도 수정 불가**
- 수정 시 → **새 템플릿 등록 + 재심사**
- 처음 등록 시 충분히 검토 (E 섹션이 검토 결과)

### G-5. 발송 시간 정책
- 알림톡(정보성): 24시간 (정보통신망법 야간 제한 미적용)
- SMS 폴백: 20시~익일 8시 자동 차단 (SOLAPI 자동 처리)

### G-6. 비용
- 알림톡: 약 9원
- SMS 폴백(90바이트): 약 22원
- LMS 폴백(2,000자): 약 33원
- 일평균 5건 × 7종 = 35건/일 → 월 1,050건 → 약 9,450원/월

### G-7. 발신번호 변경
- 070-4337-3000 등록 후 다른 번호 추가 가능 (사업자등록증 재제출)
- 변경 시 모든 템플릿 재발송에 자동 적용 (재심사 불필요)
